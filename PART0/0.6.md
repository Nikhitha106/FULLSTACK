    participant User
    participant Browser
    participant SPA as SPA JavaScript
    participant Server
    participant DB as Database

    User->>Browser: Write note and click "Save"
    Browser->>SPA: Trigger submit event
    SPA->>Server: POST /api/notes (note content as JSON)
    Server->>DB: Insert new note
    DB-->>Server: Note stored
    Server-->>SPA: 201 Created (note data as JSON)
    SPA->>Browser: Update UI with new note
